# Freelance Job Platform

## 📌 Description
A web-based freelance job platform developed with ASP.NET MVC and MariaDB.

This system allows clients to post jobs and freelancers to apply, including a full workflow from application to contract, delivery, and approval.

## 🚀 Features
- User authentication (login/register)
- Role-based system (client / freelancer)
- Job posting system
- Job application system
- Application approval/rejection
- Contract management
- Delivery & revision system
- Admin dashboard with statistics
- User management (ban/unban)

## 🛠 Technologies
- ASP.NET MVC
- C#
- MariaDB / MySQL
- Bootstrap

## 🔄 Workflow
Job → Application → Accept → Contract → Delivery → Revision / Approval
